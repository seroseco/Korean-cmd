# KCP 변경 이력

## 2.0.0 - 2026.03.16

### 변경
- KCP가 완전히 리마스터되었습니다.
- 프로젝트 명칭을 CMD 중심에서 KCP(Korean Command Prompt) 중심으로 통일했습니다.
- 대화형 프롬프트/입출력 동작을 재구성해 사용성을 개선했습니다.

### 추가
- 한글 명령어 매핑 체계를 대폭 확장했습니다.
- 주요 CMD 명령의 한글 옵션 별칭(`예: /강제, /하위`)을 추가했습니다.
- `경로`(`cd`)를 KCP 내부 명령으로 구현해 실제 작업 디렉터리 변경을 지원합니다.
- `색` 명령(색상 이름 기반)을 추가하고 색상 프리셋 명령을 지원합니다.
- `시엠디`, `창문시엠디`, `파워쉘`, `pwsh`, `터미널` 등 셸 전환/실행 명령을 추가했습니다.
- `그림판`, `메모장`, `계산기`, `작업관리자` 등 Windows 도구 실행 명령을 대량 추가했습니다.
- `깃`, `파이썬`, `도커`, `노드`, `엔피엠` 등 외부 개발 도구 실행 명령을 추가했습니다.
- 명령 목록 전용 문서 `COMMANDS.md`를 추가했습니다.

### 호환성
- v1 하위 호환 명령(`도와줘`, `색상검정`, `깨긋`, `파이썬`, `나가` 등)을 유지했습니다.

### 삭제/변경된 레거시 명령
- `블루스크린` 명령은 제거했습니다.
- v1의 `위치` 명령은 `경로`(현재 위치 출력/이동)로 통합했습니다.
- v1의 `아이피` 명령은 `네트워크`(`ipconfig`)로 대체했습니다.
- v1의 `종료` 의미(컴퓨터 종료)는 `전원`(`shutdown`)으로 분리했고, 현재 `종료`는 `taskkill` 동작으로 정리했습니다.
- `버전`은 Windows 버전(`ver`) 출력으로 정리했고, KCP 자체 버전은 시작 배너 및 변경 이력에서 관리합니다.
