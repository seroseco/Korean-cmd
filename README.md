# KCP (Korean Command Prompt)
한글로 작동하는 CMD

## ✨ 주요 특징
- 한글 명령어로 CMD 명령 실행
- 대화형 모드 지원 (`kcp.exe`만 실행)
- 명령어 옵션 한글 별칭 일부 지원 (`/강제`, `/하위` 등)
- 외부 도구 별칭 지원 (`깃`, `파이썬`, `도커` 등)
- v1 호환 별칭 일부 유지

인자 없이 실행하면 대화형 모드로 시작합니다.
종료는 `exit`, `나가기`, `나가`를 사용할 수 있습니다.

## 🧪 예시
```bat
도움말
목록
경로 ..
출력 안녕하세요
색 빨강
종료 /강제 /이름 notepad.exe
네트워크 /전체
시엠디
```

## 🧭 자주 쓰는 명령어
- 기본: `목록`, `경로`, `복사`, `이동`, `삭제`, `만들기`, `지우기`, `정리`
- 프로세스/시스템: `작업`, `종료`, `시스템`, `환경`, `도움`
- 네트워크: `확인`, `추적`, `조회`, `포트`, `주소`, `경로표`, `네트워크`
- 도구 실행: `메모장`, `그림판`, `계산기`, `작업관리자`, `제어판`, `탐색기`
- 셸 전환: `시엠디`, `창문시엠디`, `파워쉘`, `pwsh`, `터미널`
- 개발 도구: `깃`, `파이썬`, `파이썬3`, `노드`, `엔피엠`, `도커`, `쿠버`

## ℹ️ 참고
- 외부 프로그램 명령어는 해당 프로그램이 설치되어 있고 PATH에 등록되어 있어야 실행됩니다.
- 일부 시스템 도구/명령은 관리자 권한이 필요할 수 있습니다.
- 맥 OS에서는 잘 작동하지 않을수 있습니다
